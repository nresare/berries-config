# berries-config

This is the config repo for the berries cluster. The top level configuration
lives in [config.toml](./config.toml) and is transformed using
the latest version of [manifest-builder](https://github.com/nresare/manifest-builder) and
committed to [berries-manifests](https://github.com/nresare/berries-manifests).

Please make new changes via pull requests
